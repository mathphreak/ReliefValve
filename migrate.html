Migrating from Other Tools to Relief Valve

The Easiest Way

Works best if you have enough disk space on C: to hold all the games you've installed. Takes a long time.

  1. If you have enough disk space, move everything to its original location using the same tool you've been using.

  2. Set up a new Steam library following these instructions.

  3. Use Relief Valve to move all games to their proper locations.

The Simple Way

Works best if you have only a few games installed. Takes a while.

  1. Configure your external Steam library following these instructions.

    Make sure the library folder you choose is not the same exact place your previous tool has been putting games.

  2. Use Relief Valve to move all games to their proper locations.

  3. Delete the games from the previous tool's storage location.

The Efficient Way

Works best if you're confident you won't screw up and lose all your data. Not recommended.

Follow the steps that the migration wizard will follow once I add a migration wizard.

The Lazy Way

Works best if you aren't in a hurry.

Wait for me to add a migration wizard that does all this for you.
