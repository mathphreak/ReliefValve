Configuring Libraries in Steam

Switching from an older tool to Relief Valve? Read the migration guide instead.

Instructions

  1. Open Steam settings.

  2. Switch to the Downloads section.

  3. Click the Steam Library Folders button.

  4. Click the Add Library Folder button.

  5. Select the proper drive.

  6. Find the desired folder. If you don't already have a clean folder, you'll need to make a new one.

  7. Click the Select button.
